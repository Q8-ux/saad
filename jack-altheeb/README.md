# جاك الذيب 🐺

لعبة ويب جماعية خليجية أصلية من 2 إلى 4 لاعبين، تعمل بنظام الغرف ورمز الدخول.

## المزايا

- إنشاء غرفة خاصة برمز من 5 خانات.
- دخول 2–4 لاعبين أونلاين.
- نرد وأدوار وصيد الخصوم وخانات أمان ومسار نهاية.
- تصميم عربي RTL متجاوب مع الجوال.
- شخصيات خليجية كوميدية أصلية وغير منسوخة من أي عمل تلفزيوني.
- قابلة للتثبيت كتطبيق ويب PWA.

## التشغيل محليًا

```bash
npm install
npm start
```

ثم افتح:

```text
http://localhost:3000
```

## النشر على Render

- المستودع: `Q8-ux/saad`
- Root Directory: `jack-altheeb`
- Build Command: `npm install`
- Start Command: `npm start`

## ملاحظة

الغرف محفوظة في ذاكرة الخادم في النسخة الحالية، لذلك تختفي عند إعادة تشغيل الخادم. يمكن إضافة Redis أو Supabase في النسخة التجارية.
