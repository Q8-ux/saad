# جاك الذيب 🐺

لعبة ويب خليجية حماسية تدعم اللعب الفردي ضد الذيب الآلي، الوضع التعليمي، والغرف الجماعية من 2 إلى 4 لاعبين.

## المزايا

- لعب فردي مباشر ضد خصم آلي.
- وضع تعليمي يشرح الرمية والحركة والصيد ويقترح أفضل دبوس.
- غرف خاصة أونلاين برمز من 5 خانات.
- نرد وأدوار وصيد الخصوم وخانات أمان ومسار نهاية.
- تصميم عربي RTL بخط Cairo ورسومات نيون خليجية.
- متجاوبة مع الجوال وقابلة للتثبيت كتطبيق ويب PWA.

## التشغيل محليًا

```bash
npm install
npm start
```

ثم افتح `http://localhost:3000`.

## النشر على Render

- المستودع: `Q8-ux/saad`
- Root Directory: `jack-altheeb`
- Build Command: `npm install`
- Start Command: `npm start`

الغرف محفوظة في ذاكرة الخادم في النسخة الحالية، لذلك تختفي عند إعادة تشغيل الخادم.
