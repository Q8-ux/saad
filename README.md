# Smart Forms AI

نظام عربي تجريبي لإدارة المخاطبات الرسمية، بواجهة حديثة ومتجاوبة مستوحاة من التطبيقات الحكومية.

## المزايا

- لوحة تحكم وملخص يومي
- إنشاء معاملة جديدة
- رفع الخطاب الأصلي بصيغة PDF أو صورة
- الخطابات الواردة
- إعداد الردود الرسمية
- مرفقات الرد
- المراجعة والاعتماد
- مرفقات المراجعة
- الأرشيف والبحث
- التقارير ومؤشرات الأداء
- إعدادات النظام
- واجهة عربية RTL بخط Cairo
- تصميم متجاوب للجوال والكمبيوتر

## التشغيل

افتح ملف `index.html` مباشرة في المتصفح.

## النشر على GitHub Pages

1. افتح المستودع في GitHub.
2. انتقل إلى **Settings**.
3. اختر **Pages**.
4. من **Build and deployment** اختر **Deploy from a branch**.
5. اختر الفرع `main` والمجلد `/root`.
6. اضغط **Save**.

> النسخة الحالية واجهة أمامية تجريبية. قاعدة البيانات، تسجيل الدخول، OCR والذكاء الاصطناعي الحقيقي تحتاج إلى Backend وربط API.

# مشروع جاك الذيب 🐺

تمت إضافة لعبة الويب الجماعية **جاك الذيب** داخل المجلد:

```text
jack-altheeb/
```

اللعبة تعمل بخادم Node.js وSocket.IO، وتدعم غرفًا خاصة من 2 إلى 4 لاعبين.

## إعدادات نشر جاك الذيب على Render

- Root Directory: `jack-altheeb`
- Build Command: `npm install`
- Start Command: `npm start`

يوجد ملف `render.yaml` في جذر المستودع لتسهيل إنشاء خدمة Render وربط النشر التلقائي بتحديثات GitHub.
